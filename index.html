What is prism?

It's some CSS

I like the refracted-light look that's used by some movies and games (Lux from League of Legends, for example) and I wanted to see if it could be replicated in css.

Apparently it can, but it does demolish mobile batteries

How is it built?

  • Nothing but CSS.
  • DOM nodes are semantic, there are no .background or .container superfluous elements.
